Marketplace

The Marketplace is a shared hub where educators and students discover, publish, and organize scientific models, educational videos, data sets, and animated characters.

Navigation

The left sidebar organizes content into personal and public sections via a tree view:

Personal Workspace

SectionDescription
My ModelsAll models you have created, including drafts and published ones
PersonalPrivate models visible only to you
PublishedModels you have made public on the Marketplace
DraftsWork-in-progress models not yet published
FavoritesModels you have starred for quick access
LibraryPublic models you have picked/saved for personal use
DeletedSoft-deleted models that can be recovered

Public Marketplace

SectionDescription
ModelsBrowse all publicly shared scientific models
VideosEducational video content uploaded by the community
Data SetsCSV/JSON data files for model–data comparison
CharactersAnimated sprite assets for use in Body shapes

Within Models, Videos, and Data Sets, content can be filtered by Education Level and Science categories using the tree sub-nodes.

Creating a Model

Click the Create Model button in the top toolbar to start a new blank model. This opens the editor where you build your simulation. Once saved, the model appears in your Personal section.

Publishing

To share a model with the community:

  1. Open the model card's context menu or the edit popup
  2. Set the visibility to Public
  3. The model immediately becomes discoverable in the public Marketplace

You can revert a model to Private at any time to remove it from public listings.

Model Cards

Each model in the Marketplace is displayed as a card tile showing:

  • A 16:9 thumbnail preview (auto-generated or custom-uploaded)
  • The model title and description
  • Education level and science category pills
  • Like count and creator information

Card Actions

ActionDescription
OpenClick the card to open the model in the editor
LikeToggle the like heart on a model to show appreciation
FavoriteStar a model to add it to your Favorites section
PickSave a public model to your personal Library
EditOpen the metadata popup to change title, description, or thumbnail
DeleteMove the model to the Deleted bin (recoverable)

Editing Model Metadata

The edit popup allows you to modify:

  • Title — A descriptive name for the model
  • Description — A brief explanation of the scientific concept
  • Thumbnail — Upload a custom preview image
  • Education Level — Select from available levels (e.g., Elementary, Secondary, University)
  • Science — Categorize by subject (e.g., Physics, Chemistry, Biology, Mathematics)

Social Features

FeatureDescription
LikesShow appreciation for a model; like count is displayed on the card
FavoritesQuick-access bookmarks in your personal sidebar
LibraryPick public models to study and reference later
NotificationsReceive updates when others interact with your models

User Profile

Access your profile from the user menu in the top-right corner. The profile lets you configure:

  • Display name — Your public name shown on model cards
  • Country — Your location for community context
  • Language — Interface language (English or Portuguese)

Recovering Deleted Models

When you delete a model, it moves to the Deleted section. From there you can:

  • Recover — Restore the model to your personal workspace
  • Permanently delete — Remove the model irreversibly
