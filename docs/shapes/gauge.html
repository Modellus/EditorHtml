Gauge Shape

The Gauge shape displays a circular dial that visually represents a term value, useful for dashboard-style model presentations.

Screenshot: Gauge shape on the board

Overview

A Gauge shape renders a circular or semi-circular meter that fills proportionally to a bound term value. It provides an intuitive, at-a-glance visualization of quantities like speed, temperature, or progress within defined bounds.
