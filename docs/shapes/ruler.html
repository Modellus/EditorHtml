Ruler Shape

The Ruler shape provides a measurement tool for distances on the board, useful for comparing model outputs with physical measurements.

Screenshot: Ruler shape on the board

Overview

A Ruler shape places a measurement tool on the board that displays the distance between two points. It can be used to measure distances in the referential coordinate system, helping students compare simulated positions with expected values or real measurements.
