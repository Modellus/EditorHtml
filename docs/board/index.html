Board

The Board is the free-form canvas editor of Modellus. You place shapes anywhere, write the mathematics in Expression shapes, and watch tables, charts, and animations react as the model plays.

Overview

Every element on the board is a shape: expressions define the model, and the remaining shapes display or control it. The board is the default editor for Modellus models; for document-style models built as a vertical sequence of blocks, see the Notebook.

Adding Shapes

Click a shape button in the top toolbar and then click and drag on the board to draw the shape at the size you want:

  • A plain click without dragging creates nothing — drag to give the shape a size.
  • A very small drag falls back to the shape's recommended default size.
  • Press Esc or click the button again to cancel draw mode.

The table button opens a dropdown to choose between the Table and the Initial Values table. Positional shapes (Body, Point, Vector, Line, Arc) are added from a Referential, which provides their coordinate system.

Top Toolbar

  • Menu — Open, import, export, and save models
  • Undo / Redo — Also available as Ctrl+Z / Ctrl+Y
  • Shape buttons — Arm draw mode for each shape type
  • Select — Tree view of all shapes on the board for selecting them by name
  • Collaborators — Shows who else is viewing or editing the model in real time

Board Toolbar

Click an empty spot on the board to open the board toolbar. It edits the model itself rather than a shape:

  • Name — The model title
  • Theme — Mid School or University visual style
  • Background — Background color and background picture

Player Toolbar

The toolbar at the bottom drives the simulation:

  • Independent variable — Click the variable name (e.g. t) to rename it, choose the iteration term, set the number of cases (up to 9), precision, angle unit, and toggle auto play.
  • Start / Step — The domain start and integration step.
  • End — Where the simulation stops, with an option to run without limit.
  • Playback — Play/pause, stop, step backward/forward, replay, and a playhead slider to scrub through iterations.
  • Grid — Toggle snap-to-grid and set the grid size.

Saving and Offline Use

  • Signed-in users' models save to their account and appear in the Community personal workspace; changes are also synchronized in real time with collaborators.
  • Without an account you can still use the full editor and import and export model files, but cloud save is unavailable.
