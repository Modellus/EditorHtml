Modellus Documentation

Everything you need to build, simulate, and analyze mathematical models. Learn about each shape type, the expression language, data analysis tools, and more.

Shapes

Every element on the Modellus board is a shape. Each shape type has unique capabilities.

Expression

Write equations and define the mathematics of your model.

Table

Display computed values, enter data, select cells for regression.

Chart

Visualize data series, regression curves, and outliers.

Body

Animated characters that move according to your equations.

Point

A positioned dot on the board, driven by expressions.

Line

Connect two points or draw directional segments.

Arc

Draw arcs and angles between directions.

Vector

Display arrows with magnitude and direction from equations.

Slider

Interactive control to adjust parameters in real time.

Gauge

Circular dial displaying a term value visually.

Value

Show a numeric value label for any term.

Media

Embed images and backgrounds on the board.

Text

Add rich text annotations and labels.

Ruler

Measure distances on the board canvas.

Protractor

Measure and display angles interactively.

Referential

Coordinate system for positioning child shapes.

Question

Create assessment questions within a model.

Data Analysis

Modellus includes tools for statistical analysis of data collected in tables.

Regression

Fit linear or quadratic curves to your data points.

Outliers

Detect and visualize outlier data points in charts.

Marketplace

Discover, publish, and manage scientific models, videos, data sets, and animated characters.
